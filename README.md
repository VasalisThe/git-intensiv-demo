# Guess Number Game (Угадай число)

Текстовый консольный проект на чистом Python для тренировки **Git**

Репозиторий специально сделан простым и "прозрачным", чтобы его было удобно:

- **клонировать**;
- **менять** код в небольших шагах;
- **создавать ветки** под фичи;
- **делать коммиты** с осмысленными сообщениями;
- **решать конфликты** (если вы их специально спровоцируете);
- и в целом использовать как учебный репозиторий на Git-интенсиве.

Это текстовая (консольная) игра "Угадай число", разбитая на несколько файлов с
использованием ООП.

---

## 1. Описание игры

Компьютер загадывает число в заданном диапазоне (по умолчанию от 1 до 100),
игрок пытается его угадать. После каждой попытки игра подсказывает:

- если введённое число **меньше** загаданного — выводит `"Моё число больше."`;
- если **больше** — `"Моё число меньше."`;
- если число угадано — поздравляет игрока и показывает, за сколько попыток это произошло.

Игра полностью текстовая: всё происходит в терминале.

---

## 2. Для чего этот репозиторий

Этот репозиторий задуман как **учебный текстовый репозиторий для Git**:

- код достаточно простой, чтобы его мог понять начинающий разработчик;
- структура проекта чуть ближе к "реальной": есть `src`, есть `tests`,
  есть отдельные модули и класс;
- любой шаг (изменили строку, добавили метод, поправили README) легко оформить
  как отдельный коммит.

---
